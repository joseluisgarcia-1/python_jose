Operadores:

- suma = +
- resta = -
- multiplicación = *
- división = /, con este obtenemos el resultado de una división es decir si por ejemplo divido 21 % 5 el resultado es 4.2
- modulo = %, con este obtenemos el resto de una división es decir si por ejemplo divido 21 % 5 el resultado es 1
- // = con este obtenemos la parte entera de una división es decir si por ejemplo divido 21 % 5 el resultado es 4
- potencia = **, con esto podemos realizar potencias
python en cuanto a operaciones matemáticas obedece a las reglas matemáticas de la precedencia de operadores, es decir, primero los parentesis, luego potencias, luego
